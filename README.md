# Assignment 4 for Noroff

This repo was a assignment under the course Java Fullstack at Noroff 2022.

## Usage

Install dependencies

'''bash
npm install
'''

Run locally on port 4200. (-o automatically opens default browser)

'''bash
ng serve -o
'''
